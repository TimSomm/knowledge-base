# Obsidian shortcuts for MacOS
---

## Essentials

| keys      | description                                          |
| :-------- | :--------------------------------------------------- |
| ⌘ + n     | _Make new note_ |
| ⌘ + shift + n   | _Create note in new pane_ |
| ⌘ + e   | _Toggle edit/preview mode_ |
| ⌘ + o   | _Open quick switcher_ |
| ⌘ + shift + &#8595;   | _Copy line down_ |
| ⌘ + shift + &#8593;   | _Copy line up_ |
